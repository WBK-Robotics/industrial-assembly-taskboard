Assembly Guide

Welcome

The industrial assembly challenge taskboard is a modular benchmark developed at the wbk Institute of Production Science to evaluate and compare robotic assembly strategies. Each module focuses on a distinct industrial skill—from simple peg‑in‑hole insertions to belt tensioning or compliant elastic deformation.

This online documentation collects every set of step‑by‑step instructions you need to 3D‑print, source hardware, and assemble the taskboard and all of its interchangeable sub‑tasks.

How to use this guide

  1. Print or purchase the required components listed for each module.

  2. Follow the illustrated assembly sequence described in the corresponding page.

  3. Mount the finished module onto the taskboard base or use it stand‑alone for experiments.

Tip

Start with assembly_instructions_taskboard to build the base board, then add the other skill modules in any order.
